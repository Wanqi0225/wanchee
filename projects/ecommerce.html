Beauty Products E-Commerce

Full-stack Web Application Demo

Project Description

This is a beauty products e-commerce website developed as part of my Diploma assignment while learning the MVC architecture. The website allows users to browse products, search, register, login, manage their profiles, and place orders.

My Responsibilities

  • Homepage: Designed layout and product display.
  • Search Page: Implemented product search functionality using Java and JSP.
  • Login & Register: Developed user authentication system.
  • User Profile: Implemented “My Account” page with profile details and logout functionality.

Project Video

Home Page

The homepage features an auto-sliding banner at the top to showcase promotions. Below, users can browse featured products. At the bottom, a shark button allows users to quickly scroll back to the top for easy navigation.

Search Page

Users can search for products directly through the navigation bar. If no matching products are found, a ‘No Results Found’ message is displayed to inform the user.

Login & Register

Users can create a new account through the registration page and log in with their credentials. The system ensures secure authentication before accessing user-specific features.

User Profile

Users can view and update their personal information, change their password, and securely log out of their account.

Tech Stack

Java JSP MySQL JDBC GlassFish HTML/CSS/JS Servlets Netbeans IDE
