VR Fire Safety Simulator

A virtual reality simulator that immerses users in a realistic fire emergency scenario, teaching them how to respond safely and effectively. Users can interact with objects in the environment, make decisions in real-time, and learn proper fire safety protocols.

Only the features I developed are shown in this video.

My Responsibilities

  • Designed VR environment and interactive objects
  • Implemented user interactions and scenario triggers
  • Integrated VR headset controls for immersive experience
  • Tested and iterated simulation for realism and usability

Confectionery Clash: Kiki vs Johnny

In this game, Player 1 (Kiki) needs to escape a maze within a limited time, while Player 2 (Johnny) sets traps or obstacles to prevent the escape. The game emphasizes strategy, quick decision-making, and interactive gameplay, with a procedurally generated maze to increase replayability.

My Responsibilities

  • Designed game concept, mechanics, and levels.
  • Built 3D environments and objects in Unity.
  • Implemented gameplay logic and interactive features using C#.
  • Created UI elements and menus.
  • Performed testing, debugging, and optimization of the entire project.

Tech Stack

Unity C# PlayFab Photon PUN2 3D Game Development UI / Animations Single Developer Project

Shrouded Echoes

Shrouded Echoes is a team-based 3D horror investigation game developed using Unity. The project focuses on immersive storytelling, environmental exploration, and interactive gameplay to deliver a narrative-driven user experience.

My Responsibilities

  • Designed and implemented the in-game tutorial to guide players through core mechanics.
  • Created the story prelude and early chapters (Chapter 1), establishing narrative flow and pacing.
  • Built and arranged 3D environments in Unity to support exploration and storytelling.
  • Used Unity Timeline to control cutscenes, story sequences, and player experience.
